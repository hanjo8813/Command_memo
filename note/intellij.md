# 인텔리제이

## 단축키

### window

|단축키|설명|
|---|---|
|Alt + Enter|수정 도우미|
|Ctrl + Space|추천 도우미|
|Shift + Shift|패키지 검색|
|Ctrl + W|드래그블록 범위 확장|
|Ctrl + Shift + W|드래그블록 범위 축소|
|Ctrl + Alt + L|코드 정렬|
|Shift + Tab|왼쪽 정렬|
|Ctrl + Shift + Alt + T|리팩토링 메뉴(중요)|
|Ctrl + Alt + V|변수 리팩토링|
|Alt + I (원래 insert)|오버라이딩 단축키(난 i로 바꿈)|
|Ctrl + Shift + A|액션 검색|
|Ctrl + F10|실행서버 옵션|
|Alt + Shift + U|Camel Case 변경|
